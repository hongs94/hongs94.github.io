Attention-LSTM을 활용한 환율 변동 추이 분석

환율 (USD, CNY, JPY, EUR) 4종의 데이터와 여러 독립변수들의 데이터를 수집해 환율 변동 추이 분석을 진행했습니다. XGBoost, LSTM, Attention-LSTM 모델을 생성해 학습하고 1일 후 환율 예측을 하였습니다.

학습한 모델의 성능을 비교하고 1일 후 예측값을 누적해 그래프로 시각화 했습니다.

Shadcn 과 V0를 사용해 UI를 개발하고 대시보드를 생성하고 양쪽 사이드바에 채팅 기능과 모델 및 통화 선택 기능을 추가했습니다. 중앙은 Chart.js를 사용해 그래프로 환율 데이터를 시각화 했습니다.

  • 사용한 모델: XGBoost, LSTM, Attention-LSTM
  • 사용한 기법: Rolling Window를 활용한 시계열 데이터 학습
  • 옵티마이저: Adam을 활용한 모델 파라미터 최적화

MBC 아카데미에서 진행한 2번째 프로젝트로 개발 후 AWS를 통해 배포까지 완료했습니다.
