Python을 활용한 부분적 PMS 생성

Python과 Flask를 활용한 부분적 PMS를 생성 했습니다. 로그인, 회원가입 기능과 실시간 업데이트로 회사 직원들과 업무 일정 공유가 가능하도록 설계되었고, 저는 업무 일정일정 페이지를 담당해 생성형 AI와 Vscode를 사용해 코딩하였습니다.

기능 구현: 회원정보 관리, 메인 페이지, My 페이지, 팀 페이지, 알림 페이지, 프로젝트 관리, 업무 일정, FAQ, 자유 게시판

개선이 가능한 부분:

  • 업무일정 페이지의 실행 속도: CRUD 기능은 잘 구현했으나 속도가 느린점이 아쉬움
  • 코드 최적화: 팀원이 각 페이지를 맡아 작업하여 병합했으나 리팩토링 또는 리뉴얼을 통해 최적화 작업 가능
  • UI 디자인: Figma를 사용해 UI를 디자인하고 HTML, CSS, JS를 사용해 구현했으나 v0 또는 shadcn을 사용해 개선 가능

MBC 컴퓨터 아카데미에서 진행한 첫 프로젝트로 개발을 경험한 뜻 깊은 시간이었습니다. 결과물은 다소 미흡할지라도 기능 구현을 성공적으로 마침에 의의를 두었습니다.
